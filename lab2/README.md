# Lab 2 – Bayesian Inference

## Overview

This project explores **Bayesian inference** for estimating unknown quantities based on noisy observations.

## Problem Description

A target emits signals detected by sensors (radar).  
The goal is to estimate the position of the target using probabilistic methods.

## Objectives

- Apply Bayes theorem in practice
- Work with likelihood functions
- Combine prior knowledge with observed data
- Estimate parameters under uncertainty

## Approach

- Define a probabilistic model of observations
- Compute likelihood based on sensor readings
- Combine with prior distribution
- Estimate posterior distribution

## Key Concepts

- Prior, likelihood, posterior
- Uncertainty modeling
- Noise in measurements
- Maximum a posteriori (MAP) estimation

## Conclusion

Bayesian methods allow incorporating prior knowledge and handling uncertainty explicitly, making them suitable for real-world noisy data problems.
