# Statistical Machine Learning – Projects

This repository contains a set of projects developed as part of a course in **Statistical Machine Learning**.  
The focus of the course is on probabilistic modeling, Bayesian inference, and practical implementation of machine learning methods.

## Course Overview

The course introduces core concepts of statistical learning with a strong emphasis on:

- Bayesian interpretation of probability
- Probabilistic models and uncertainty
- Gaussian models and multivariate distributions
- Linear regression (classical and Bayesian)
- Monte Carlo methods
- Model inference and prediction

All exercises are implemented in **Python using Jupyter Notebooks**, combining theory with practical experimentation.

## Topics Covered

### 1. Monte Carlo Methods
Numerical methods for approximating values using random sampling:
- Estimation of π
- Volume estimation in higher dimensions
- Random sampling techniques
- Convergence and variance analysis

### 2. Bayesian Inference
Probabilistic reasoning under uncertainty:
- Bayes theorem
- Prior, likelihood, posterior
- Parameter estimation
- Working with noisy observations

### 3. Regression Models
Modeling relationships between variables:
- Linear regression (MLE)
- Bayesian linear regression
- Posterior predictive distribution
- Uncertainty in predictions
