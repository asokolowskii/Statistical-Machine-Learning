# Lab 3 – Regression Models

## Overview

This project focuses on **linear regression models**, including both classical and Bayesian approaches.

## Objectives

- Model relationships between variables
- Estimate model parameters from data
- Understand differences between deterministic and probabilistic approaches

## Implemented Methods

### Linear Regression (MLE)
Parameters are estimated by minimizing squared error (least squares method).

### Bayesian Linear Regression
Instead of single values, parameters are modeled as probability distributions.

### Posterior Predictive Distribution
Predictions are made by integrating over parameter uncertainty.

## Key Concepts

- Least squares estimation
- Gaussian noise assumption
- Parameter uncertainty
- Predictive distributions

## Conclusion

Regression models are fundamental tools in machine learning.  
Bayesian approaches extend classical regression by explicitly modeling uncertainty.
